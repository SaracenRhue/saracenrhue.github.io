## Get started

Create a new project

```bash
npx create-electron-app@latest my-app
```

> nodejs and npm must be installed
{: .prompt-info }

Run the app

```bash
cd my-app
npm start
```

## Build the app

Build the app for your current platform

```bash
npm run make
```
